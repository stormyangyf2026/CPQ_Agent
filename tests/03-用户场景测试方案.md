## 场景3: 价格反向匹配 🆕

👤 销售工程师小张 → 🎯 预算有限，找最优方案

### 操作步骤

| 步骤 | 用户输入 | 页面交互 | 预期 Agent 响应 |
|------|---------|---------|----------------|
| 3.1 | "客户预算只有 25000，帮我找匹配的储能方案" | 用户消息气泡（蓝色右对齐） | Agent 调用 reverse_match_price(target_price=25000) |
| 3.2 | 等待 | 左侧橙色 AI 头像 + 脉冲点动画 + "正在分析..." | Agent 搜索产品 → 获取定价 → 排序推荐 |
| 3.3 | 查看结果 | 响应以结构化文本呈现 | 显示：✅ 找到 X 个预算内方案 |
| 3.4 | 选择方案 | 用户回复 "选第二个" | Agent 展开该方案的详细 BOM + 定价 |

### 预期输出格式

```
正在为您匹配 25000 元预算内的储能方案...

🔍 共搜索到 8 个产品，其中 3 个在预算内

📊 最佳匹配：
1. EVE-LVI-5.0 — ¥4,500 (预算内，剩余 ¥20,500)
2. EVE-CTP-30K — ¥32,000 (超出预算 ¥7,000)

💡 推荐：EVE-LVI-5.0 低压壁挂储能，完全在预算内
```

### 测试脚本

```python
#!/usr/bin/env python3
"""场景3: 反向匹配"""
import requests, json, sys

def scenario_3():
    r = requests.post('http://localhost:58100/agent/chat',
        json={'messages': [{'role':'user','content':'预算25000，推荐储能方案'}]},
        stream=True, timeout=90)
    
    text = ''
    events = set()
    for line in r.iter_lines(decode_unicode=True):
        if line.startswith('event: '):
            events.add(line[7:])
        elif line.startswith('data: '):
            d = json.loads(line[6:])
            text += d.get('content','')
    
    assert len(text) > 15, f'响应太短'
    keywords = ['匹配','价格','预算','方案','推荐']
    found = [k for k in keywords if k in text]
    assert len(found) >= 2, f'关键词不足: {found}'
    assert 'status' in events and 'message_delta' in events and 'done' in events
    print(f'✅ 场景3 PASS: 反向匹配 ({len(text)}字符, {len(found)}关键词)')
    return True

if __name__ == '__main__':
    try: scenario_3(); sys.exit(0)
    except Exception as e: print(f'❌ {e}'); sys.exit(1)
```

## 场景4: 方案对比 🆕

👤 销售主管老王 → 🎯 两个产品之间做取舍

### 操作步骤

| 步骤 | 用户输入 | 预期 Agent 响应 |
|------|---------|----------------|
| 4.1 | "帮我对比一下 modelId=2019 和 modelId=2009 这两个电芯" | Agent 调用 compare_solutions(a=2019, b=2009) |
| 4.2 | 等待处理 | Agent 同时获取两个产品的详情 + BOM + 定价 |
| 4.3 | 查看对比结果 | 返回结构化对比表 + 差异分析 + 推荐结论 |

### 预期输出格式

```
📊 方案对比：LF280K vs LF50K

| 对比项 | LF280K | LF50K | 差异 |
|--------|--------|-------|------|
| 价格 | ¥820 | ¥158 | 差价 ¥662 |
| 容量 | 280Ah | 50Ah | ⚠️ 230Ah |
| 循环次数 | 8000 | - | LF280K 更耐用 |

💰 价格差：¥662
💡 推荐：如果用于大型储能电站，推荐 LF280K（容量大、循环长）
```  

### 测试脚本

```python
#!/usr/bin/env python3
"""场景4: 方案对比"""
import requests, json, sys

def scenario_4():
    r = requests.post('http://localhost:58100/agent/chat',
        json={'messages': [{'role':'user','content':'对比 modelId=2019 和 modelId=2009 两个产品'}]},
        stream=True, timeout=90)
    
    text = ''
    for line in r.iter_lines(decode_unicode=True):
        if line.startswith('data: ') and 'content' in line:
            text += json.loads(line[6:]).get('content','')
    
    assert len(text) > 50, f'响应太短: {len(text)}'
    keywords = ['对比','差异','价格','A','B','推荐','差价']
    found = [k for k in keywords if k in text]
    assert len(found) >= 2, f'对比信息不足: {found}'
    print(f'✅ 场景4 PASS: 方案对比 ({len(text)}字符, {len(found)}关键词)')
    return True

if __name__ == '__main__':
    try: scenario_4(); sys.exit(0)
    except Exception as e: print(f'❌ {e}'); sys.exit(1)
```
